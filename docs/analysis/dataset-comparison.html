PL Court Name Inconsistencies

The Polish dataset contains court name typos where the space between "Apelacyjny" and the preposition is missing:

  • "Sąd Apelacyjnyw Warszawie" should be "Sąd Apelacyjny w Warszawie" (1,076 + 141 = 1,217 total)
  • "Sąd Apelacyjnywe Wrocławiu" should be "Sąd Apelacyjny we Wrocławiu" (899 + 302 = 1,201 total)
  • "Sąd Apelacyjnyw Szczecinie" should be "Sąd Apelacyjny w Szczecinie" (584 + 80 = 664 total)
  • "Sąd Apelacyjnyw Białymstoku" should be "Sąd Apelacyjny w Białymstoku" (270 + 16 = 286 total)

In the Court Distribution chart (Section 4), these variants have been merged to show the true distribution.

MMethodology

Sampling Strategy

Both datasets were sampled at 6,050 documents each to enable fair comparison:

  • UK Dataset (JuDDGES/en-court-raw): 6,050 English criminal appellate judgments from the England and Wales court system, spanning 2003-2024. The majority originate from Crown Court appeals heard by the Court of Appeal (Criminal Division).
  • PL Dataset (JuDDGES/pl-appealcourt-criminal): 6,050 Polish criminal appellate court (Sąd Apelacyjny) judgments from 2012-2025, distributed across all 11 Polish appellate court jurisdictions.

Analysis Methods

  • Text length: Characters counted from raw text; words tokenized by whitespace splitting; sentences split using language-appropriate rules (spaCy/nltk).
  • Vocabulary: Token-level analysis using lowercased whitespace tokenization on the full corpus. Type-Token Ratio (TTR) computed as unique_tokens / total_tokens. Stopwords excluded from domain word rankings.
  • Topics (UK): Keyword-based extraction using regex pattern matching for common criminal law terms. Documents can match multiple topics. Percentages indicate prevalence in the corpus.
  • Topics (PL): From the structured "keyword" metadata field in the dataset. Only 76.7% of PL cases have keyword annotations.
  • Court name normalization (PL): Variants with missing spaces (e.g., "Apelacyjnyw") were merged with their correct counterparts for the court distribution chart.
  • Sentence length: Average words per sentence computed as total_words / total_sentences per document, then averaged across the corpus.

Limitations

  • Cross-language vocabulary comparisons are inherently limited. Polish is a highly inflected language with many more unique word forms than English, inflating unique token counts and TTR.
  • Sentence splitting accuracy varies between languages; Polish legal text with extensive abbreviations (art., ust., pkt.) can produce artificially short segments.
  • UK topic extraction via keyword matching may undercount nuanced topics. PL keywords are human-annotated but not available for all cases.
  • The UK dataset has longer temporal coverage (22 years) vs PL (14 years), which may affect temporal trend comparisons.
